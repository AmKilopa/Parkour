# Parkour

Паркур от первого лица на Godot 4. Цель — пройти уровень как можно быстрее, используя продвинутые паркур-механики.

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| Мышь | Обзор |
| Space | Прыжок |
| C | Слайд |
| Ctrl | Крауч (присесть) |
| Shift | Захват края / Long Jump заряд |
| ЛКМ | Выстрел из дробовика |
| R | Перезапуск уровня |
| Q/E | Наклон камеры вверх/вниз |
| Esc | Пауза |

## Механики

**Базовые:**
- **Coyote Time** — небольшое окно после схода с платформы, когда ещё можно прыгнуть
- **Jump Buffer** — прыжок заранее зажмётся в буфер, сработает при приземлении

**Продвинутые:**
- **Wall Run** — бег по стенам при достаточной скорости, с пониженной гравитацией
- **Wall Jump** — прыжок от стены с разгоном
- **Ledge Grab** — если бежишь к краю и не дотягиваешься, зажми Shift — персонаж схватится за выступ. Потом нажми Space чтобы перелететь наверх
- **Long Jump** — заряженный дальний прыжок. У края зажми Shift (заряд), потом отпусти чтобы прыгнуть далеко вперёд
- **Bunny Hop** — стрейф + разворот мыши для набора скорости в воздухе
- **Slide Jump** — прыжок из слайда с сохранением импульса
- **Ramp Launch** — рампа разгоняет вверх при съезде с достаточной скоростью
- **Landing Speed** — приземление по направлению движения сохраняет скорость

**Оружие:**
- Дробовик — дробь с разбросом и отдачей
- Взрывающиеся бочки — стреляй чтобы подорвать

## Система очков

Старт: **100 очков**. Тратится за использование механик:
- Обычный прыжок: -1
- Стейф/банни хоп прыжок: -1
- Wall Jump: -2
- Long Jump: -2
- Ledge Grab: -4
- Выстрел: -5

Чем меньше потрачено очков — тем чище прохождение.
